# Print summary diagnostics

Displays a human-readable summary of data cube diagnostics.

## Usage

``` r
# S3 method for class 'summary_cube_diagnostics'
print(x, ...)
```

## Arguments

- x:

  A `summary_cube_diagnostics` object returned by
  `summary(diagnose_cube(...))`.

- ...:

  Additional arguments passed to other methods (currently unused).

## Value

The input object `x`, returned invisibly.
