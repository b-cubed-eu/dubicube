# Print a cube_rule object

Prints a concise summary of a diagnostic rule used by
[`diagnose_cube()`](https://b-cubed-eu.github.io/dubicube/reference/diagnose_cube.md).
The output shows the rule identifier, cube dimension, and threshold used
to determine severity.

## Usage

``` r
# S3 method for class 'cube_rule'
print(x, ...)
```

## Arguments

- x:

  An object of class `cube_rule`.

- ...:

  Additional arguments passed to other methods (currently unused).

## Value

The input object `x`, returned invisibly.
